# PIXEL BLOG 🕹️

A high-performance, ultra-optimized pixelated blog with a "Classic Arcade" aesthetic.

## Features
- **CRT Scanline Effect**: Authentic retro monitor feel with subtle flickering.
- **Pixel-Perfect UI**: Custom CSS borders and "Press Start 2P" typography.
- **Fluid Animations**: Framer Motion powered entry and interactive effects.
- **Gamified Elements**: "Insert Coin" interaction with reactive high-score tracking.
- **Optimized for Vercel**: Ready for one-click deployment.

## Tech Stack
- React + TypeScript
- Vite
- Framer Motion
- Lucide React

## Setup & Run
1. Navigate to the folder:
   ```bash
   cd pixel-blog
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Deployment
Push this folder to GitHub and link it to Vercel. It will automatically detect the Vite configuration.
